# Intelligent OCR Text Extraction and Error Detection System

## Overview
This project is a **web-based OCR application** built with Flask that extracts text from uploaded images and PDF files. It not only retrieves text but also highlights errors, detects spelling mistakes, and provides a detailed summary of the content. The system gives users insights into text quality and confidence scores for each OCR operation.

---

## Features

- Upload images or PDF files for text extraction
- Displays extracted text with **errors highlighted in red**
- Shows a **summary of total words, total lines, and OCR confidence scores**
- Detects **spelling mistakes** and provides suggestions
- User-friendly web interface for quick and easy operation
